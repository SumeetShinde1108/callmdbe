## ğŸ“ Glossary

**Terms used across documentation:**

- **SuperAdmin** - Platform-wide administrator role
- **Agent** - Organisation administrator (1 per org)
- **SuperUser** - Same as Agent (role value in database)
- **Regular User** - Basic user with limited access
- **Organisation** - Multi-tenant entity
- **Permission** - Granular access right (24 total)
- **Agent Assignment** - Process of making user an agent
- **Permission Bypass** - SuperAdmin automatic access
- **Role Sync** - Automatic role upgrade/downgrade
